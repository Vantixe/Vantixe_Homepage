# Vantixe Homepage

**Procurement Excellence. Delivered.**

Dual-domain website for Vantixe Advisory Limited - procurement consulting (vantixe.com) and AI-powered technology products (vantixe.ai), served from a single Next.js application.

---

## Stack

- **Next.js 15** (App Router) + TypeScript
- **Tailwind CSS 4** for styling
- **Framer Motion** for animations
- **Vercel** for hosting
- **Cloudflare** for DNS

## Domains

| Domain | Content |
|--------|---------|
| www.vantixe.com | Consulting - advisory services, founder profile, service detail pages |
| vantixe.ai | Technology - TPRM platform, Negotiation Agent, Category Strategy |

## Local Development

```bash
npm install
npm run dev       # http://localhost:4000
```

## Deploy

```bash
npx vercel --prod
```

See [DEPLOYMENT.md](DEPLOYMENT.md) for full infrastructure details, DNS config and page inventory.
